## ⚙️ Setup Instructions

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/Khalipha-Samela/Portfolio.git
   ```

2. Navigate into the project folder:

   ```bash
   cd Portfolio
   ```

3. Set up EmailJS configuration:

   * Create a file: `assets/js/config.js`
   * Copy contents from `config.example.js`
   * Add your EmailJS credentials

4. Open `index.html` in your browser

---

## 🔐 Environment Configuration

This project uses a `config.js` file for sensitive data (EmailJS keys).

⚠️ This file is ignored via `.gitignore` and not included in the repository.

Use `config.example.js` as a reference.
